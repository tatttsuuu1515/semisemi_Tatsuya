  • 西講中について

    大久保町は旧西国街道沿いに開けた町で昔は宿場町として栄え、今も旧本陣跡が残っている歴史ある町です。 その大久保町の鎮守、住吉神社の祭りは春と秋の年2回行われており100年以上歴史がある祭りです
    春祭りは5月の第3日曜日とその前日、秋祭りは10月の第3日曜日とその前日に行われ、宵宮と本宮の2日間にわたって開催されます。
    祭りは氏子の集まりである西と東の住吉講がそれぞれ布団太鼓やだんじり、 獅子舞等を有し、宮入・奉納をしています。
    私たちは西住吉講、略称は西講中で、地元では「にしんじょ」と呼んでいます。
    ここでは、西講中の祭りについて紹介していきます。
